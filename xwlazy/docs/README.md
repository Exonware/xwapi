# xwlazy documentation

Documentation for the `exonware-xwlazy` package.

## What it covers

- On-demand installs for missing imports inside a configured package scope (with import-name → PyPI-name maps).
- Optional lazy loading for heavier dependencies.
- One-line activation in `__init__.py` with a safe fallback if xwlazy is not installed.
- PEP 668-safe behavior and optional audit artifacts (lockfile, SBOM).
